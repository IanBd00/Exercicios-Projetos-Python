# Exercicios-Projetos-Python

Este repositório reúne materiais, exercícios de fixação e projetos desenvolvidos durante o processo de estudo da linguagem Python por meio da plataforma Digital Innovation One (DIO).

O conteúdo apresentado tem como objetivo consolidar os fundamentos da linguagem Python, bem como aplicar conceitos teóricos em situações práticas, por meio da resolução de problemas e do desenvolvimento de projetos progressivos.
